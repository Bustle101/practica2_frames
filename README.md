Мини веб служба с модулями

1) Ядро
- Контракт как основа для создания модулей
- Контейнер для хранения модулей
- Валидация самих модулей и их зависимостей

2) Модули
- logger
- validation
- report
- export

Реализован локальынй сервер, для запуска и проверки работы модулей. 

logger работает на каждом этапе работы приложения

Запрос отправляется на сервер, где происходит обработка, к которой подключаются модули. Validation проверяет тело запроса на корректные данные. Report дополняет тело запроса подробностями. Export создает файл и сохраняет основные данные.

запрос -> validation -> report -> export

Запуск приложения:
```
npm start
```
Тесты: 
```
npm test
```
